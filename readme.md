## 🛠️ How to Use

1. **Read the notes** — Open any `notes.md` in VS Code and use the **Live Preview** extension (or any Markdown previewer) to see the formatted content.
2. **Run the code** — Execute the `.js` files with Node.js:
   ```bash
   node Day1/basics.js
   ```

## 📋 Prerequisites

- [Node.js](https://nodejs.org/) installed
- A code editor (VS Code recommended)
- VS Code **Live Preview** extension (for rendering Markdown notes)
